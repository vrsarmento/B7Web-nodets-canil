# B7Web-nodets-canil

Projeto do Curso B7Web - Módulo Node.js + Typescript

### Pré-requisitos globais:

`npm i -g nodemon typescript ts-node`

### Instalação

`npm install` ou `yarn install`

### Para executar o projeto

`npm run dev` ou `yarn dev`
